# Gestión de estudiantes en Python

Programa en consola hecho en Python para guardar y administrar estudiantes usando un archivo JSON.

## Funciones

* Agregar estudiantes
* Mostrar lista
* Modificar datos
* Eliminar estudiantes
* Buscar por carrera
* Ver un reporte básico

## Cómo ejecutar

```bash
python gestion_estudiantes.py
```
